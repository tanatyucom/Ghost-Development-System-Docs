# Information Package Template

## Purpose

Use this template to share project state between AI assistants, humans, and
future Command Center workflows.

An Information Package summarizes current project state, evidence, recent
artifacts, open issues, and recommended next action. It does not replace the
repository as the source of truth.

## Review Checklist

- Project Summary is clear.
- Current Status is current.
- Current Focus is explicit.
- Active Repository and boundaries are named.
- Related Rules and Templates are named.
- Recent Decisions include evidence.
- Open Issues include next actions.
- Recent Artifacts have paths.
- Recommended Next Q is present or intentionally omitted with reason.
- Notes do not replace source-of-truth repository documents.
