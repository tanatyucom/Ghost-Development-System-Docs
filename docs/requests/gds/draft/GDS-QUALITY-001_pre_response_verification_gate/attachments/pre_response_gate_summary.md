# Pre-Response Verification Gate Summary

## Added Standard

This Q adds a final response quality gate after artifact generation and before
the AI delivers the final response.

## Canonical Artifacts

- Workflow: `docs/workflow/pre_response_verification_gate.md`
- Checklist: `templates/ai_response_checklist_v2.md`
- Response verification template: `templates/response_verification_checklist.md`

## Key Rule

The final response must match actual work, verification results, user-requested
format, repository scope, and commit / push reality.
