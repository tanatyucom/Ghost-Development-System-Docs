# Ghost Research Registry

## Purpose

This registry lists Ghost Research assets so humans, AI, and future Repository
Intelligence tooling can discover research topics, decisions, candidate links,
and review status.

The registry is an index. The individual Research asset remains the canonical
detailed record.

## Fields

| Field | Meaning |
| --- | --- |
| Research ID | Stable `GR-XXX` identity. |
| Title | Human-readable topic. |
| Category | Research category. |
| Status | Lifecycle state. |
| Adoption Decision Summary | High-level summary; idea-level details live in the asset. |
| Priority | Adoption priority. |
| Difficulty | Implementation difficulty estimate. |
| Related Candidates | Future Candidate or Platform Candidate references. |
| Related Evidence | Evidence references. |
| Related ADR | ADR references. |
| Path | Canonical Research asset path. |
| Last Reviewed | Last review date. |
