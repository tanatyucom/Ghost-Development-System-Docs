# Update Policy

新しい運用ルールを追加する場合は

既存ファイルへ追加するか、

新しいRule Documentとして分離するかを検討する。

One File One Theme

One Folder One Responsibility

を維持する。

Rulesは

実績を基に改善する。

机上の理論ではなく、

実際の運用で効果が確認されたものを標準化する。
