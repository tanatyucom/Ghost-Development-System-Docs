# Philosophy

## Purpose

This folder preserves the North Star, collaboration philosophy, and experience
principles that must survive across GDS generations.

Architecture explains structure. Workflow explains sequence. Philosophy
explains why the structure and sequence exist.

## Contains

- `north_star.md`: the durable collaboration goal for GDS.
- `human_ai_collaboration_model.md`: how humans and AI share judgment,
  approval, recommendation, and execution boundaries.
