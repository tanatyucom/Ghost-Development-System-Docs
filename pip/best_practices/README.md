# PIP Best Practices

## Purpose

Best Practice entries capture reviewed patterns that should be reused across Q files, completion reports, review workflows, and future projects.

## Expected Content

- Best Practice ID
- Title
- Problem it prevents
- Recommended practice
- Required artifacts
- Related cases
- Related rules
- Validation notes

## Scope

Best Practices are not higher priority than official rules. If a practice must become mandatory, promote it to `docs/rules/` through review.
